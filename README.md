# Invitación de Boda Cristian & Karen

Proyecto Vite + React + TypeScript con Tailwind y framer-motion para la invitación web de Cristian David & Karen Yadira.

## Scripts

- `npm run dev` — servidor de desarrollo
- `npm run build` — build de producción
- `npm run preview` — previsualización del build
- `npm run lint` — linting

## Notas

- El favicon y las imágenes sociales usan `/favicon.svg` con el corazón “CyK`”.
- Audio de fondo: `src/assets/Cancion_de_Fondo.mp3` (autoplay puede requerir interacción según el navegador).
